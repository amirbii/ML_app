# 🧠 Streamlit Persian ML App

این پروژه یک رابط کاربری تعاملی برای تحلیل داده‌ها و اجرای مدل‌های یادگیری ماشین است که با استفاده از **Streamlit** و با رابط فارسی طراحی شده است.

## 📌 امکانات فعلی:

- 📁 **بارگذاری داده** از فایل CSV یا لینک مستقیم (Github/Kaggle)
- 📊 **نمایش آماری داده‌ها** شامل تعداد، نوع ستون‌ها، مقادیر گمشده و توصیف آماری
- 🧹 **حذف داده‌های پرت** با استفاده از روش‌های:
  - میانگین و انحراف معیار (STD + Mean)
  - بازه بین چارکی (IQR)
  - الگوریتم Local Outlier Factor (LOF)
- 🔁 **نرمال‌سازی داده‌ها** با StandardScaler یا MinMaxScaler
- ➗ **تقسیم داده به آموزش/تست**
- 🤖 **انتخاب مدل یادگیری ماشین** از بین:
  - Logistic Regression
  - Support Vector Machine (SVM)
  - K-Nearest Neighbors (KNN)
  - Decision Tree
- ⚙️ **تنظیم پارامترهای مدل‌ها**
- 🧪 **آزمایش مدل با تصویر تست**
- 🧾 **تولید کد نهایی (در دست توسعه)**
